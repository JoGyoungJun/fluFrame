# EPIC-open-source-readiness

Status: Draft
Layer: presentation
Source GDD: `design/gdd/flutter-app-framework.md`
Priority: MVP

## Goal

Make FulFrame understandable and contributable as an open-source project before
the first public release.

## Scope

### In Scope

- README.
- License decision.
- Contributing guide.
- Code of conduct.
- Issue and pull request templates.
- Public roadmap.
- Codex for OSS application notes.

### Out Of Scope

- Community moderation automation.
- Sponsorship setup.
- Package publishing.
