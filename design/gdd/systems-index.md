## Status Values

- `Draft`: being designed.
- `In Review`: design is mostly complete and under review.
- `Approved`: ready to convert into epics and stories.
- `Implemented`: implemented and verified.

## Layer Values

- `foundation`: infrastructure, project setup, save/load, input, core architecture.
- `core`: primary gameplay systems.
- `feature`: supporting gameplay features.
- `presentation`: UI, visuals, audio, game feel, polish.
